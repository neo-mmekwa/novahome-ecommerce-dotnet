# NovaHome - E-Commerce Platform

## Overview
A full-stack e-commerce web application built with ASP.NET, C#, SQL Server, and Power BI, designed with scalable architecture and real-world payment simulation and invoicing workflows.

## Tech Stack
- ASP.NET / .NET
- C#
- SQL Server
- Power BI

## Features (Planned / In Progress)
- User authentication
- Product catalog
- Shopping cart
- Order & payment workflow
- Invoice generation
- Analytics & reporting

## Project Status
🚧 In active development – built incrementally by feature.

## Architecture Notes
- Incremental database design
- Snapshot-based financial records
- Scalable schema
